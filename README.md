# 📊 Sales Performance Dashboard using Microsoft Excel

## 📌 Project Overview

This project presents a Sales Performance Dashboard built using Microsoft Excel.  
The objective of this project is to clean raw sales data, analyze key business metrics, and create an interactive dashboard using Pivot Tables and Charts.

This project demonstrates practical data analysis skills including data cleaning, aggregation, and visualization.

---

## 🎯 Project Objectives

- Clean and organize raw sales dataset
- Handle missing values and duplicate records
- Correct inconsistent data formats
- Calculate Total Revenue and Units Sold
- Analyze performance by Region and Category
- Identify Monthly Sales Trends
- Create visual dashboards for business insights

---

## 📂 Dataset Description

The dataset includes the following columns:

- Order_ID  
- Order_Date  
- Region  
- City  
- Category  
- Product_Name  
- Units_Sold  
- Unit_Price  
- Total_Sales  
- Salesperson  

The dataset intentionally contained:

- Missing values  
- Duplicate entries  
- Inconsistent text formatting (e.g., "west" vs "West")  
- Incorrect calculations  
- Mixed date formats  

All issues were identified and resolved using Excel data cleaning techniques.
